# Buck Converter

Buck Converter pcb kartı linkteki [video](https://www.youtube.com/watch?v=n6VyZIZuCj0) izlenerek tamamlanmıştır.

- LM2596 voltaj regülatörü kullanılmıştır.
- Pcb kartı 2 katmanlıdır ve 37x39 mm boyutundadır.
